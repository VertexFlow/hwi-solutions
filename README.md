# DSA Questions

Curated problem sets for competitive programming practice.

## Structure

```
├── easy/      (13 questions)
├── medium/    (8 questions)
└── hard/      (9 questions)
```

## Each Folder Contains

- `questions.md` - Problem statements
- `analysis.md` - Patterns and key learnings
- `assets/questions.pdf` - Original problem PDF
- `{index}_{slug}/` - Per-question folder with:
  - `solution.java`
  - `solution.cpp`
  - `notes.md`

## Quick Navigation

| Difficulty | Questions | Topics |
|------------|-----------|--------|
| Easy | 13 | Arrays, DP basics, Greedy |
| Medium | 8 | Advanced DP, Trees, Segment Trees |
| Hard | 9 | Grid DP, Tree DP, Complex Scoring |
