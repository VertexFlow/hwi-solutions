# Analysis of Medium DSA Questions

## Questions Overview

This set contains 8 medium difficulty problems:

1. Maximum Modulo Sum - DP with remainders
2. Dual Timeline - Greedy scheduling
3. Minimum Cost Swaps - Graph/cycle decomposition
4. Vineyard Planting - DP with state pruning
5. Mosaic Tile Optimization - 1D DP per row
6. Aquarium Fish Display - DP sequence merging
7. Tree Plucking - Greedy leaf removal
8. Recaman Queries - Segment tree with math

## Problem Types and Patterns

### Dynamic Programming (4 problems)
- Q1 (Maximum Modulo Sum) - 1D DP tracking remainders
- Q4 (Vineyard Planting) - 2D DP with state pruning
- Q5 (Mosaic Tile Optimization) - 1D DP for tiling
- Q6 (Aquarium Fish Display) - 2D DP for sequence merging

### Greedy Algorithms (3 problems)
- Q2 (Dual Timeline) - Assign to team with less time
- Q3 (Minimum Cost Swaps) - Cycle decomposition approach
- Q7 (Tree Plucking) - Remove cheapest leaf first

### Data Structures (1 problem)
- Q8 (Recaman Queries) - Segment tree for range queries

## Key Concepts Required

1. Dynamic Programming
   - State definition and transitions
   - Space optimization
   - State pruning for complex DP

2. Greedy Algorithms
   - Proving optimality
   - Priority queue usage

3. Graph Algorithms
   - Cycle decomposition
   - Tree manipulation

4. Data Structures
   - Segment trees
   - Priority queues
   - Hash sets/maps

5. Number Theory
   - Prime checking
   - Recaman sequence
   - Fortunate numbers

## Recommendations for Beginners

Before attempting similar problems:

1. Master basic DP patterns
   - 1D DP (fibonacci, house robber)
   - 2D DP (edit distance, LCS)
   - State space reduction

2. Learn greedy proof techniques
   - Exchange argument
   - Matroid theory basics

3. Practice graph problems
   - Tree traversals
   - Cycle detection

4. Study common data structures
   - Segment trees
   - Binary indexed trees
   - Priority queues

5. Review number theory basics
   - Primality testing
   - Prime factorization
